A stack of translucent layers, translated on the z-axis with sibling-index(), driven by registered @property plus CSS trigonometric functions and perspective/perspective-origin.
  • Register numeric custom properties so browser can interpolate them
  • Map pointer position into rotateX and rotateY values
  • Let CSS own the transform math instead of re-rendering DOM every frame
This layered stack blends 3D transforms with pointer math; each layer translates along z so the card feels volumetric while staying rooted in CSS.

Depth comes from two tiny rules

  • perspective and perspective-origin react to current tilt, so depth follows pointer
  • Each layer uses translateZ() with sibling-index() to create stack spacing
  • Same visual trick, much cheaper than repainting separate scenes in JS

registered @property

Differently than normal custom properties, the ones registered via @property at-rule are transitionable and animatable, allowing to calculate interpolation between all the specified values.
