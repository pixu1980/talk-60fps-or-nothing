When the trigger is the viewport itself, the browser can drive the timeline natively without waking JS on every intersection. 

.scroll-view-card {
  animation: reveal-and-elevate 0.8s cubic-bezier(0.34, 1.56, 0.64, 1) both;
  animation-range: entry 35% cover 70%;
  animation-timeline: view(inline);
}

@keyframes reveal-and-elevate {
  from {
    opacity: 0;
    transform: scale(0.85) translateY(8rem);
    filter: blur(8px);
  }
}
